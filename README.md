# zentho-hotel-project
The Zentho Hotel Website:

Welcome to the official repository for The Zentho Hotel, a luxury hotel website project developed for our ICT course. This website showcases a fictional heritage luxury hotel located in Islamabad, Pakistan.

Project Overview:

This project is a complete multi-page website designed to provide a realistic booking experience for users. It features a modern, responsive design with a "Heritage Luxury" theme, combining traditional aesthetics with modern web standards.

Key Features:

Multi-Page Navigation: Home, About Us, Rooms, Services, Book Now, and Contact Us.

Responsive Design: Works on desktops, tablets, and mobile devices.

Dynamic Booking System: JavaScript-based booking form that calculates prices automatically.

Checkout Simulation: A checkout page that carries over booking details (room type, price) using LocalStorage.

Contact Form: A styled contact form for guest inquiries.


Technologies Used:

HTML: For the structure and semantic layout.

CSS: For styling, grid layouts, flexbox, and the custom "Brown & Gold" luxury theme.

JavaScript: For the booking logic, price calculation, and form handling.


IDE Used:

Visual Studio Code.

Project Structure:

home.html - The main landing page.

aboutus.html - Information about the hotel's history and team.

rooms.html - Showcase of available rooms (Single, Twin, Double).

services.html - Details on amenities like the Pool, Gym, and Dining.

booknow.html - The reservation page with price calculation.

checkout.html - Payment simulation page.

contactus.html - Contact form and details.

style.css - The master stylesheet for the entire website.

/images - Folder containing all assets (logos, room photos, etc.).

How to Run This Project

Clone the repository to your local machine:

git clone [https://github.com/Muhammad-Saad06/zentho-hotel-project.git]


Open the folder in your file explorer.

Double-click on home.html to open the website in your browser.
